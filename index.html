верстка сайтов

Кроссбраузерная, валидная, качественная

Верстка сайта начинается с подробного изучения дизайна. На основе макета с помощью языка HTML создается разметка будущего сайта. Затем с помощью каскадных таблиц стилей CSS сайт обретает свою индивидуальность. Верстка проверяется на валидность, то есть на отсутствие ошибок и соответствие стандартам. Готовый сайт открывается в популярных браузерах чтобы проверить корректное отображение и функционирование.
